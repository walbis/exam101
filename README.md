## İçerik

### Network (CCNA)
- OSI ve TCP/IP modelleri
- Port numaraları
- Subnetting hesaplamaları
- VLAN, STP, OSPF
- Cisco komutları
- 15+ Flashcard

### Bilgi Güvenliği
- CIA Üçlüsü
- OWASP Top 10 (2021)
- Saldırı türleri (Network, Web, Sosyal Mühendislik)
- Şifreleme (Simetrik/Asimetrik/Hash)
- Kimlik doğrulama faktörleri
- 20+ Flashcard

### Temel SQL
- SELECT, WHERE, JOIN
- NULL kontrolü (IS NULL vs = NULL)
- GROUP BY ve HAVING
- Aggregate fonksiyonlar
- Alt sorgular
- 20+ Flashcard

### Scripting/OS
- Linux Bash komutları
- Dosya izinleri (chmod, 644, 755)
- Süreç yönetimi
- PowerShell cmdlet'leri
- Bash vs PowerShell karşılaştırma
- 20+ Flashcard
