## 1.1 OSI Model (10 Soru)

### Soru 1.1.1
**Konu:** OSI Katmanları
**S:** OSI modelinde kaç katman vardır?
**C:** 7 katman

### Soru 1.1.2
**Konu:** OSI Katmanları
**S:** OSI modelinde 7. katman hangisidir?
**C:** Application (Uygulama)

### Soru 1.1.3
**Konu:** OSI Katmanları
**S:** OSI modelinde 1. katman hangisidir?
**C:** Physical (Fiziksel)

### Soru 1.1.4
**Konu:** OSI Veri Birimleri
**S:** Transport katmanındaki veri birimine ne denir?
**C:** Segment

### Soru 1.1.5
**Konu:** OSI Veri Birimleri
**S:** Network katmanındaki veri birimine ne denir?
**C:** Packet

### Soru 1.1.6
**Konu:** OSI Veri Birimleri
**S:** Data Link katmanındaki veri birimine ne denir?
**C:** Frame

### Soru 1.1.7
**Konu:** OSI Cihazları
**S:** Router hangi OSI katmanında çalışır?
**C:** Katman 3 (Network)

### Soru 1.1.8
**Konu:** OSI Cihazları
**S:** Switch hangi OSI katmanında çalışır?
**C:** Katman 2 (Data Link)

### Soru 1.1.9
**Konu:** OSI Cihazları
**S:** Hub hangi OSI katmanında çalışır?
**C:** Katman 1 (Physical)

### Soru 1.1.10
**Konu:** OSI Protokolleri
**S:** HTTP hangi OSI katmanında çalışır?
**C:** Katman 7 (Application)

## 1.2 TCP/IP Model (5 Soru)

### Soru 1.2.1
**Konu:** TCP/IP Katmanları
**S:** TCP/IP modelinde kaç katman vardır?
**C:** 4 katman

### Soru 1.2.2
**Konu:** TCP/IP Katmanları
**S:** TCP/IP modelinin 4 katmanını yukarıdan aşağıya sırala.
**C:** Application, Transport, Internet, Network Access

### Soru 1.2.3
**Konu:** TCP/IP - OSI Karşılığı
**S:** TCP/IP'nin Application katmanı OSI'nin hangi katmanlarına karşılık gelir?
**C:** Katman 7, 6, 5 (Application, Presentation, Session)

### Soru 1.2.4
**Konu:** TCP/IP - OSI Karşılığı
**S:** TCP/IP'nin Internet katmanı OSI'nin hangi katmanına karşılık gelir?
**C:** Katman 3 (Network)

### Soru 1.2.5
**Konu:** TCP/IP - OSI Karşılığı
**S:** TCP/IP'nin Network Access katmanı OSI'nin hangi katmanlarına karşılık gelir?
**C:** Katman 2 ve 1 (Data Link, Physical)

## 1.7 VLAN (6 Soru)

### Soru 1.7.1
**Konu:** VLAN Tanımı
**S:** VLAN ne işe yarar?
**C:** Broadcast domain'i mantıksal olarak ayırır

### Soru 1.7.2
**Konu:** Varsayılan VLAN
**S:** Cisco switch'lerde varsayılan VLAN numarası kaçtır?
**C:** 1

### Soru 1.7.3
**Konu:** Native VLAN
**S:** Native VLAN ne demektir?
**C:** Tag'siz (untagged) trafiğin gittiği VLAN

### Soru 1.7.4
**Konu:** Access Port
**S:** Access port ne işe yarar?
**C:** Tek bir VLAN'a bağlı port, son kullanıcı cihazları için

### Soru 1.7.5
**Konu:** Trunk Port
**S:** Trunk port ne işe yarar?
**C:** Birden fazla VLAN trafiğini taşır (switch-switch arası)

### Soru 1.7.6
**Konu:** Inter-VLAN
**S:** VLAN'lar arası iletişim için ne gereklidir?
**C:** Router veya Layer 3 Switch

## 1.11 Static Route (3 Soru)

### Soru 1.11.1
**Konu:** Static Route Komutu
**S:** Static route komut yapısı nedir?
**C:** ip route [hedef-network] [subnet-mask] [next-hop]

### Soru 1.11.2
**Konu:** Default Route
**S:** Default route nasıl yazılır?
**C:** ip route 0.0.0.0 0.0.0.0 [next-hop]

### Soru 1.11.3
**Konu:** Static Route Örnek
**S:** 192.168.2.0/24 ağına 10.0.0.2 üzerinden ulaşmak için komut?
**C:** ip route 192.168.2.0 255.255.255.0 10.0.0.2

## 1.12 OSPF (5 Soru)

### Soru 1.12.1
**Konu:** OSPF Tipi
**S:** OSPF ne tür bir routing protokolüdür?
**C:** Link-state routing protokolü

### Soru 1.12.2
**Konu:** OSPF Metric
**S:** OSPF'in metric'i nedir?
**C:** Cost (bant genişliğine dayalı)

### Soru 1.12.3
**Konu:** Area 0
**S:** OSPF'te Area 0 nedir ve önemi ne?

### Soru 1.12.4
**Konu:** Hello/Dead Timer
**S:** OSPF'te varsayılan Hello ve Dead timer değerleri nedir?
**C:** Hello: 10 saniye, Dead: 40 saniye

### Soru 1.12.5
**Konu:** Cost Formülü
**S:** OSPF Cost formülü nedir?
**C:** Cost = 100 Mbps / Bandwidth
